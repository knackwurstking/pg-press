# PG Press Handover Document

## Project Overview

- This project uses golang, (a-h) templ for html templating
- Styling is mostly done using the ui library inside the assets directory "internal/assets/assets/css/ui.min.css"
- Generating .templ file using the `templ` tool, (ex.: `templ generate`) or just use `make generate`
