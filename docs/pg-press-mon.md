# Performance Monitoring Tool

**Location:** `/cmd/pg-press-mon`

The *Performance Monitoring Tool* is a terminal‑based user interface (TUI) that measures the latency of HTTP endpoints, providing instant feedback and the ability to rerun tests interactively.

* Key features
  * Measure endpoint latency accurately
  * Interactive TUI for visualizing results and re‑executing measurements

> **Note:** This is a conceptual idea; development has not yet begun.
